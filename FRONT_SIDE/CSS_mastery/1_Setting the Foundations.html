HTML includes a rich variety of meaningful elements, such as
  1. h1, h2, and so on
  2. strong and em
    1. strong
    2. em
  3. blockquote and cite
    1. is blockquote
    2. is cite
  4. abbr, acronym, and code
    1. is abbr
    2. is acronym
    3. is code
  5. fieldset, legend, and label
  6. caption, thead, tbody, and tfoot
As such, it is always a good idea to use an appropriate meaningful element where one exists.
